# Multi-Jurisdiction Compliance Response Matrix

## Overview

This repository contains a structured legal and regulatory analysis of data deletion requests received by ShopGhana from customers in Ghana, Germany (EU), and California (USA).

The project evaluates compliance obligations under:

- Ghana’s Data Protection Act, 2012 (Act 843)  
- The EU General Data Protection Regulation (GDPR)  
- The California Consumer Privacy Act (CCPA) and California Privacy Rights Act (CPRA)  

It provides a comparative compliance framework, response timelines, retention justifications, and jurisdiction-specific response strategies.

## Purpose

This repository demonstrates how an e-commerce platform operating globally must handle cross-border data subject requests while complying with multiple privacy regimes simultaneously.

It highlights:

- Differences in deletion rights  
- Varying statutory response deadlines  
- Exemption handling  
- Documentation and audit requirements  
- Enforcement risk exposure
